# Task Manager (Java + JDBC + MySQL)

## 📌 Features
- Add Task
- View Tasks
- Update Task
- Delete Task

## 🛠️ Tech Stack
- Java
- JDBC
- MySQL

## 🚀 How to Run
1. Setup MySQL database
2. Update DB credentials in DBConnection.java
3. Run MainApp.java

## 📚 Concepts Used
- OOP
- JDBC
- CRUD Operations
